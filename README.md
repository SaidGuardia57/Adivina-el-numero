Este proyecto consiste en un juego en Python donde el usuario debe adivinar un número secreto generado aleatoriamente.

---

## Descripción

El programa genera un número secreto dentro de un rango dependiendo de la dificultad seleccionada.  
El jugador tiene un número limitado de intentos para adivinarlo.

Después de cada intento, el programa indica si el número secreto es **mayor** o **menor** que el número ingresado.

---

## Funcionalidades

-  Generación de número aleatorio
-  Selección de dificultad:
  - Fácil (1–10)
  - Medio (1–20)
  - Difícil (1–50)
-  Repetición del juego
-  Límite de 5 intentos
-  Mensajes de ayuda (mayor / menor)
-  Finaliza cuando se adivina o se acaban los intentos

---

## Trabajo en equipo

Este proyecto fue desarrollado utilizando GitHub con ramas (branches).

Cada integrante trabajó en una parte específica:

- dificultad.py → selección de dificultad  
- numero_secreto.py → generación del número  
- intentos.py → control de intentos  
- repetir_juego.py → repetir partida  
- mensajes.py → mensajes del juego  
- main.py → integración del proyecto
